Class: Windows

browser.Windows()

The window storage is used to save outer window IDs mapped to weak references of Window objects.

Usage:

let wins = new browser.Windows();
wins.set(browser.outerWindowID, window);

...

let win = wins.get(browser.outerWindowID);
