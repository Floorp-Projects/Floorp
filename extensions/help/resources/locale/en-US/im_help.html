What Is Instant Messenger?

Netscape Instant Messenger is an easy way to communicate instantly with your friends who are online. You can:

  • Send instant messages to an online buddy. These messages are different from email, because if your buddy has Instant Messenger open on his or her machine, your message will appear immediately on your buddy's screen.
  • Create a "buddy list" of online friends who you want to receive your instant messages. Instant Messenger helps you locate your buddies so you can add them to your list.
  • Control who communicates with you through the buddy controls feature.
  • Chat with groups of friends and family members in your own private chat room.
  • Control who communicates with you through the buddy privacy feature.
  • Find your buddies by email address.
  • Warn users who send you offensive or annoying messages.

Setting Up

To get started quickly, use the Instant Messenger Setup screens.

  1. If the Netscape Instant Messenger Buddy List isn't already on your screen, open the Tasks menu and choose Instant Messenger, or click the Buddy List tab on the Sidebar.
  2. Click Setup.
  3. Follow the steps in the Instant Messenger Setup screens. Before you can use Instant Messenger, you must register a screen name. The Setup screens lead you through the process.

Registering New Screen Names

When you set up Instant Messenger for the first time using the Instant Messenger Setup screens, you register and select a screen name and password. In subsequent Instant Messenger sessions, you can register multiple screen names to use with the Netscape Instant Messenger service.

To register and select a new screen name and password:

  1. Go to the Netscape Instant Messenger home page, http://aim.aol.com/aimnew/Aim/register.adp. The Instant Messenger web page appears on your screen, allowing you to register.
  2. Type your screen name, password (twice for confirmation), and email address in the appropriate fields.
  3. After you have completed the registration process, use your new screen name and password in the sign-on screen.

Signing On to Netscape Instant Messenger

Before you can use Instant Messenger to send a message or to chat, you need to start the program and sign on.

To start and sign on to Netscape Instant Messenger:

  1. Display the Netscape Instant Messenger Sign On screen to start the program. You can display the screen in three different ways:
    • Open the Tasks menu and choose Instant Messenger. Or:
    • Click the Instant Messenger icon in the taskbar at the bottom of the Netscape 6 window. Or:
    • Click the Buddy List tab in the Sidebar.
  2. Type your screen name in the Screen Name field.
  3. Type your password in the Password field. If you want to save your password for your next session, select "Save password."
  4. Click Sign On, or press Enter.

Tip: If you want Netscape Instant Messenger to start when you start Netscape 6, select "Sign on at launch" on the sign-on screen.

Using Multiple Screen Names

If you have registered multiple screen names, you can sign on with the name of your choice. To sign on with a new screen name that you have registered:

  1. Display the Sign On screen.
  2. Select the current screen name that is displayed in the Screen Name field, and type your new screen name over that.
  3. Type the password for your new screen name in the Password field.
  4. Click Sign On.

Setting Preferences

You can set preferences for Instant Messenger to meet your particular needs.

If you have previously used AOL Instant Messenger, and you have installed Netscape 6 on your machine, you will have to select your preferences again. Preferences, unlike Buddy Lists, do not transfer automatically from AOL Instant Messenger to Netscape 6.

You can set preferences for such things as:

  • Displaying graphical smiles in messages
  • Indicating your online status in Netscape Mail
  • Specifying who can contact you
  • Turning on sounds to play when a you receive a message or a buddy signs on
  • Editing your Away messages

You set your preferences using the Instant Messenger Preferences panels. To access the panels:

  • Open the Edit menu, choose Preferences, and then select one of the Instant Messenger categories. You may need to click the widget (triangle) next to the words Instant Messenger to see all of the categories. Or:
  • Click the Preferences button in the toolbar of the standalone Instant Messenger window.

Specifying Connection Settings

If you access the Internet through an Internet Service Provider, you most likely will not need to make any changes to the Connection preferences panel. If you access the Internet through a local area network firewall (for example, at your workplace), then your network administrator may need to configure the proxy section of the Connection preferences panel.

Here are descriptions of the items in the Connection panel:

Server Host

This field contains the default Internet address of the Netscape Instant Messenger server. Under normal circumstances, you do not need to change this field. If your proxy server needs to resolve the host name, you may need to put the IP address instead of the name of the server in this field.

Server Port

This field contains the default server port number to be used on the Netscape Instant Messenger server. Under normal circumstances, you do not need to change this field.

Connect using proxy

If you click this button, the Netscape Instant Messenger software activates the proxy server controls.

Proxy Server Host

Supply the name of the proxy server.

Proxy Server Port

Supply the port number to be used on the proxy server.

Protocol

Specify whether your proxy server uses the SOCKS 4, SOCKS 5, or HTTPS protocol.

Authentication Username

Supply the authentication username required by the proxy.

Authentication Password

Supply the authentication password required by the proxy.

About the Buddy List

When you sign on to Instant Messenger, your Buddy List appears. Three groups are automatically created for you in your Buddy List window: Buddies, Family, and Co-Workers. Add your buddies to any of these groups or create additional groups.

There are two tabs in your Buddy List: List Setup and Online.

  • List Setup lets you add buddies to the list and create groups for them. For example, you can create groups for specific interests such as Golf Pals, College Roommates, Neighbors, and so forth, then add your buddies to them. The List Setup tab also lets you delete buddies and groups.

    • In the List Setup tab:

    • Each group appears as a folder.
    • The buddies in each group appear as branches from the folder. You must expand the group to see buddies in that group. To expand the group, click the triangle next to the folder so that the triangle points downward. Or, select the name of the group and press Enter.
    • You cannot see which of your buddies is online when you display the List Setup tab. Click the Online tab to see which buddies are online.

  • Online shows you when your buddies are available. Whenever your buddies sign on, you can see them in your Buddy List and send them messages instantly.

    • In the Online tab:

    • Groups in the Buddy List window are initially shown expanded, displaying all your buddies in each group. You can switch the display mode between expanded and collapsed by clicking the small triangle to the left of each group name; or, select (highlight) the group name and press Enter.
    • Each group has a number in parentheses to the right of the group name to indicate how many people in the group are online out of the total number of buddies in the group. For example, (3/9) means that three of the nine buddies in the group are currently online.
    • Buddy names in bold indicate the buddies who signed on most recently. Buddy names in gray and italic indicate the buddies who just signed off.

Tip: Buddy lists are stored on a server rather than on your computer. That means you can access your buddy lists from anywhere, whenever you sign on to the Netscape Instant Messenger service.

Adding and Finding Buddies

If you know a friend's screen name, you can add him or her to your Buddy List immediately.

  1. In the Buddy List, click the List Setup tab.
  2. Click the Add Buddy icon. You see the Add Buddy dialog box.
  3. Type the screen name of the buddy you want to add. Remember to enter just the buddy's screen name, not the full email address. For example, johndoe, rather than johndoe@aol.com.
  4. Select the group to which you want to add the screen name.
  5. Click OK.

If you don't know a friend's screen name, Instant Messenger can help you locate your friend using his or her email address.

  1. >From the People menu, select Find A Buddy.
  2. Type your buddy's email address in the "Friend's email address" field and click Next. Netscape Instant Messenger displays all screen names for the email address you specified.
  3. Select the screen name you want to add to your list and click Add Buddy.
  4. In the Add Buddy dialog box, select the group to which you want to add the buddy, and click OK. Or, click Send IM to send a message to your buddy instantly.
  5. Click Finish when you are done.

Creating Buddy Groups

You can group your buddies to organize your buddy list. Your list comes with three pre-set groups: Buddies, Family, and Co-Workers.

To create a new group:

  1. In the Buddy List, click the List Setup tab.

  3. Type a name for the group and click OK.

To add a new buddy, follow the steps under Adding and Finding Buddies.

To open or close a group's folder, click the right triangle next to the folder. Or, select the name of the group to highlight it and press Enter.

Using the Sidebar Buddy List

If you use the Sidebar feature of Netscape 6, then you may find it convenient to include your buddy list in a Sidebar tab.

If it is not already open, open the Sidebar by clicking on its handle.


The Sidebar Buddy List may be closed, so that only its header is showing. To open it, click the tab on which the words Buddy List appear.
 

The Sidebar Buddy List lets you know when one of your buddies is online and available for a chat, even if Netscape Instant Messenger is not open.

If your buddy is online, his or her name appears in your buddy list. If your buddy is idle, his or her name is dimmed and a small clock icon appears next to the name.

To send an instant message:

  1. Double-click a buddy's name in the Sidebar Buddy List.
  2. Type your message and click Send.

Tip: On Windows and Linux systems, you can right-click a name and choose to send an instant message or a chat invitation. On Mac OS systems, use Control-click.

To add or remove buddies, click the List Setup tab to add buddies or create buddy groups.

Checking the Status of Buddies

To find out which of your buddies is online, click the Online tab in the Buddy List. If you also keep your Buddy List displayed in the Sidebar, you can check there to see who's online.

You cannot see which of your buddies is online when you have the List Setup tab displayed. Click the Online tab.

In the Online tab:

  • To expand and collapse a group, click the small triangle next to the name of the group.
  • To see how many people in a group are online, check the number in parentheses next to the group name. For example, (4/7) means that four of the seven buddies in the group are currently online
  • Names displayed in gray are idle (idle means that a person hasn't used the computer for 15 minutes but is still online). The clock icon also appears next to a name of a buddy who is idle.

In Netscape Mail:

You can also find out which of your buddies is online from within Netscape Mail. If a buddy is online, a presence icon appears next to his or her name in a message window, in the To, From, or CC fields. This icon also appears next the names of non-buddies. Non-buddies are Instant Messenger users who you have not added to your buddy list. For both buddies and non-buddies, the presence icon will show only for those users who have registered and confirmed their registration.

The presence icon next to a buddy's name disappears when he or she logs on or off. The presence icon does not change next to a non-buddy's name.

Importing and Exporting Buddy Lists

If you have used a previous version of AOL Instant Messenger, you can transfer your buddy list.

  1. Launch and sign on to AOL Instant Messenger.
  2. From the My AIM menu, choose Save Buddy List. The file is saved with the extension .blt.
  3. Sign off and quit AOL Instant Messenger.
  4. Launch and sign on to Instant Messenger in Netscape 6.
  5. From the file menu, choose Import Buddy List.
  6. Select the file you saved to import your Buddy List.

Sending an Instant Message

Follow these steps to create and send an instant message:

  1. Open the Instant Message compose window.
    • To send an instant message to someone in your Buddy List, double-click a name in the list. A window opens where you can type your message. If your buddy is online, the To field is filled in automatically. If your buddy isn't online, type your buddy's screen name in the To field.
    • To send an instant message to someone who is not in your Buddy List, click the Send IM icon in the Online tab of the Sidebar. Or, in the Instant Messenger standalone window, open the People menu and choose Send Instant Message. Type the user's name in the To field.
  2. Type your message in the compose area of the Instant Message window.
  3. Click the Send button (the flying envelope in the lower-right corner of the window), or press Enter. The message is sent.

Tip: On Windows and Linux systems, you can right-click a name and choose to send an instant message or a chat invitation. On Mac OS systems, use Control-click.

You see your message in the top frame of the message window. Your buddy's response will also appear in the top pane. You can continue typing and sending messages as long as you and your buddy want to converse.

Inviting Buddies to Chat

To invite a buddy to chat:

  1. In the Buddy List window, click the Online tab.
  2. Select the screen names of the buddies you want to invite to the chat room. To select multiple screen names, hold down the Ctrl key as you select each name. You can also select an entire group of buddies.
  3. Click the Buddy Chat icon.

    The Buddy Chat Invitation window opens. If you want to send the invitation to additional buddies, type their screen names in the Screen Names to Invite field, separated by commas.

  4. Click Invite to send the invitation.

    5. After your buddy accepts your invitation, Netscape Instant Messenger displays a message indicating your buddy has entered the room. You can start typing your chat messages in the text area after you receive the notification.

You can invite a buddy to join a chat in progress. To do so, click the Invite button on the toolbar in the Chat window.

Stopping Messages from People in the Chat Room

To stop receiving messages from one or more people in the chat room:

  1. Select the person’s screen name (Shift-click to select multiple names) in the field that lists participants in the chat room.
  2. Click Ignore. An red circle with a slash through it appears next to the selected screen name. Click Ignore again to turn the Ignore feature off.

Once you've clicked Ignore, you won't see any messages from those people whose screen names you selected. However, others in the chat room will continue to see the messages of those people.

Away Messages

 

If you need to step away from your computer but do not want to sign off, you can use an Away message to let others know that you are temporarily unavailable.

For planned absences, you can turn on your Away message to let others know when you will be back. Netscape Instant Messenger displays this message when anyone requests information about you and automatically replies with your Away message to anyone who sends you a message while you are away.

In this section:

Creating a New Away Message

Turning an Away Message On and Off

Deleting an Away Message

 

Creating a New Away Message

To create a new Away message:

  1. Open the Edit menu and choose Preferences.
  2. Under the Instant Messenger category, select Away. If no subcategories are available under Instant Messenger, click the triangle next to the words Instant Messenger to expand the list.
  3. Make sure "Auto respond with 'away' message" is selected.
  4. To create a new message, click Add Message. The New Away Message dialog box appears on your screen.
  5. Type a label for the new away message in the Label field.
  6. Type the message in the field below "Message to send when you receive a message."
  7. Click OK to save your changes and close the dialog box.

Netscape Instant Messenger adds your new message to your list of Away messages.

[ Return to beginning of section ]

 

Turning an Away Message On and Off

To turn an Away Message on or off:

  1. Click Away at the bottom of the Netscape Instant Messenger window, or select the message you want to use from the pull-down menu. To turn off instant message sounds while you are away, click "Disable sounds" from the Instant Messenger Away preference panel.
  2. To turn off your Away message, click I'm Back in the Instant Messenger window.

[ Return to beginning of section ]

 

Deleting an Away Message

To delete an Away message:

  1. Open the Edit menu and choose Preferences.
  2. Under the Instant Messenger category, select Away. If no subcategories are visible under Instant Messenger, click the triangle next to the words Instant Messenger to expand the list.
  3. Select the Away message that you want to delete.

You can also set privacy preferences to control which users can send you messages, and how much information about you other Instant Messenger users can have.

Specifying Who Can Contact You

Instant Messenger lets you control who can send you messages, and how much information about you other users can have.

  1. Open the Edit menu and choose Preferences.
  2. Under the Instant Messenger category, select Privacy. If no subcategories are available under Instant Messenger, click the triangle next to the words Instant Messenger to expand the list.
  3. Click the button next to the option that describes who you want to be able to contact you.
  4. If you selected "Allow these users," click Add Name and type the screen name of the person you want to allow to contact you. To remove someone from the allow list, click his or her screen name, then click Remove.
  5. In the lower section of the Privacy panel, choose an option from the pop-up menu for how much information users who know your email address should be able to see about you.
  6. Click OK.

[ Return to beginning of section ]

 

Blocking a User

You can block one or more users so they can't send you messages. This feature is useful if you want to block users who send offensive messages or if you simply do not want to receive messages for a while.

  1. Open the Edit menu and choose Preferences.
  2. Under the Instant Messenger category, select Privacy. If no subcategories are available under Instant Messenger, click the triangle next to the words Instant Messenger to expand the list.
  3. Select "Block all users" or "Block these users."
  4. If you selected "Block these users," click Add Name and type the screen name of the person you want to block.

To remove someone from the block list, click his or her screen name, then click Remove.

Warnings

If someone else using the Netscape Instant Messenger service sends you annoying or offensive messages, you have two ways to respond. You can issue a warning to the person or you can block the person.

In this section:

Warning a User

Viewing Someone's Warning Level

 

Warning a User

To warn a user who has sent you an offensive message, click Warn in the Instant Message window in which you received the message.

Warn imposes a warning on the sender, hampering that person's ability to send and receive messages. With frequent warnings, the user will not be able to exchange any messages with anyone. Only after a “cooling off” period can that user send and receive messages again.

When you click Warn, you can choose to impose the warning anonymously. Anonymous warnings are less severe and the user you are warning won't know who issued the warning.

Regarding warnings you send and receive:

  • You can issue only one warning for each message someone sends you.
  • If a user warns you, consider ending the conversation. Every additional message you send enables that user to warn you again.

[ Return to beginning of section ]

 

Viewing Someone's Warning Level

You can see someone’s warning level next to the person's screen name in your Buddy List window.

Alternatively, you can permanently block one or more people from sending you messages. See "Blocking a User" for more information.
