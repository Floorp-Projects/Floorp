Preferencias de apariencia: apariencia

En esta sección se explica el uso del panel de preferencias de apariencia. Si aún no ha llegado a alguno de esos paneles, siga los pasos que se indican a continuación:

  1. Abra el menú Editar y seleccione Preferencias.
  2. Haga clic en la categoría Apariencia.


El panel de preferencias de apariencia permite establecer los componentes que se abrirán al iniciar Netscape:

  • Cuando Netscape arranca, se abre: seleccione los componentes que desea abrir al iniciar Netscape.
  • Mostrar información sobre herramientas: seleccione esta opción si desea recibir sugerencias al colocar el cursor sobre las distintas partes de la interfaz de usuario de Netscape.
  • Mostrar iconos de sitios web: Seleccione esta opción si desea ver los iconos propios de las páginas web en lugar del icono de marcador . Los iconos de los sitios web aparecen a la izquierda de la barra Dirección y en la lista de marcadores.

 

Preferencias de apariencia: fuentes

En esta sección se explica el uso del panel de preferencias de las fuentes. Si aún no ha llegado al panel, siga los pasos que se indican a continuación:

  1. Abra el menú Editar y seleccione Preferencias.
  2. En la categoría Apariencia, elija Fuentes. (Si no aparece ninguna subcategoría, haga doble clic en Apariencia para expandir la lista.)


El panel de preferencias de las fuentes permite establecer el tipo y el tamaño de letra, además de la resolución de pantalla.

Nota: es posible que algunos estilos de fuente no se puedan seleccionar debido a que el idioma seleccionado no disponga de fuentes para dicho estilo.

  • Fuentes para: seleccione el juego de caracteres de un idioma. Por ejemplo, para definir fuentes predeterminadas para el juego de caracteres Occidental (Roman), elija Occidental.
    • Proporcional: seleccione si el texto proporcional debe ser serif (como Times Roman) o sans-serif (como Arial). Indique también el tamaño de fuente que desee para el texto proporcional. El texto proporcional varía en anchura, por lo que los caracteres y las letras también varían en anchura.
    • Serif: seleccione la fuente serif que desee utilizar para las páginas web.
    • Sans-serif: seleccione la fuente sans-serif que desee utilizar para las páginas web.
    • Itálica: seleccione la fuente cursiva que desee utilizar para las páginas web.
    • Fantasía: seleccione el tipo de letra de fantasía que desee utilizar para las páginas web.
    • Monospace: seleccione la fuente monospace (como Courier) que desee utilizar para las páginas web. El texto monospace es fijo en anchura, por tanto todos los caracteres o letras ocupan el mismo espacio.
    • Tamaño de fuente mínimo: seleccione el tamaño mínimo de letra que desee ver en las páginas web.
  • Permitir que los documentos utilicen otras fuentes: si está disponible, marque esta casilla de verificación para mantener los valores de fuente y tamaño de una página web, en lugar de las preferencias propias.
  • Resolución de pantalla: seleccione en la lista desplegable los puntos por pulgada (dpi) para la visualización de páginas web. Seleccione Otro para abrir el cuadro de diálogo Calibrar resolución, donde podrá calibrar la resolución midiendo la longitud de una línea que aparece en pantalla. Si aumenta la resolución de pantalla, mejorará la legibilidad de algunas pantallas.

 

Preferencias de apariencia: colores

En esta sección se explica el uso del panel de preferencias de colores. Si aún no ha llegado al panel, siga los pasos que se indican a continuación:

  1. Abra el menú Editar y seleccione Preferencias.
  2. En la categoría Apariencia, elija Colores. (Si no aparece ninguna subcategoría, haga doble clic en Apariencia para expandir la lista.)


El panel de preferencia de colores permite fijar los colores del fondo y del texto de las páginas web:

  • Texto y fondo: haga clic en los bloques de colores para seleccionar el color en el que se verán el fondo y el texto de las páginas web.
    • Utilizar valores del sistema: seleccione esta opción para usar los valores de color de su sistema.
  • Colores de los enlaces: haga clic en los bloques de colores para seleccionar el color en el que se verán los enlaces visitados y no visitados de las páginas web.
    • Enlaces subrayados: seleccione esta opción para subrayar los enlaces de las páginas web.
  • Cuando una página web proporciona sus propios colores y fondos:
    • Usar siempre los colores y el fondo que especifique la página web: permite a la web elegir los colores y los fondos.
    • Usar siempre los colores elegidos e ignorar los colores e imagen de fondo especificados: le permite a usted elegir los colores, pasando por alto los colores y la imagen de fondo de la página web.
  • Habilitar resaltado de sintaxis: seleccione esta opción para que la sintaxis HTML del origen de las páginas web se vea resaltado en determinados colores.

    Sugerencia: para ver el origen de una página web, abra el menú Ver de Navigator y seleccione Origen de la página.

 

Preferencias de apariencia: temas

En esta sección se explica el uso del panel de preferencias de temas. Si aún no ha llegado al panel, siga los pasos que se indican a continuación:

  1. Abra el menú Editar y seleccione Preferencias.
  2. En la categoría Apariencia, elija Temas. (Si no aparece ninguna subcategoría, haga doble clic en Apariencia para expandir la lista.)


El panel de preferencias de temas permite cambiar el aspecto y el estilo de Netscape, basándose en temas diferentes.

  • Temas para utilizar: elija uno de los temas instalados para cambiar el aspecto de Netscape.
    • Desinstalar [nombre del tema]: haga clic aquí para eliminar un tema seleccionado.
  • Obtener temas nuevos: haga clic en este enlace para ir a una página web en la que puede descargar más temas.

    Notas:

    • Después de aplicar un tema nuevo, es necesario salir y volver a iniciar Netscape para verlo.
    • Si desinstala un tema, no podrá volver a utilizarlo.

 

Preferencias de apariencia: idiomas/contenido

En esta sección se explica el uso del panel de preferencias de idioma y paquetes de contenido. Si aún no ha llegado al panel, siga los pasos que se indican a continuación:

  1. Abra el menú Editar y seleccione Preferencias.
  2. En la categoría Apariencia, haga clic en Idiomas/Contenido. (Si no aparece ninguna subcategoría, haga doble clic en Apariencia para expandir la lista.)


El panel de preferencias de paquetes de idioma y contenido le permite seleccionar el paquete de contenido y de idioma que desea usar en Netscape.

  • Paquetes de contenido instalados: seleccione el paquete de contenido que desea utilizar dentro de Netscape.
  • Paquetes de idioma instalados: seleccione el paquete de idioma que desea utilizar dentro de Netscape.
  • Descargar más: haga clic en esta opción para descargar más paquetes de idioma.

    Nota: para que tenga efecto el nuevo paquete de contenido debe cerrar Netscape y volver a abrirlo.
