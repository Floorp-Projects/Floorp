Los atajos de teclado de Netscape

Netscape permite usar atajos de teclado en lugar del ratón para numerosos comandos. Al usar el teclado para acceder a comandos, se suele ahorrar tiempo; además, los atajos ayudan a aquellas personas con problemas físicos, de vista o de oído a usar Netscape con mayor facilidad.

Empleo de atajos de teclado

En este documento se emplea el siguiente formato para relacionar los atajos de teclado:

Ejemplo:

Comando
Windows
Mac
Linux
Copiar
Ctrl+C
Cmd+C
Ctrl+C

Para ejecutar un comando, pulse al mismo los botones que se relacionan. Por ejemplo, para copiar en Windows, pulse Ctrl y C al mismo tiempo.

Sugerencia: el signo más (+) significa que debe pulsar al mismo tiempo las teclas que están delante y detrás del signo. Si hay más de un signo más (como Ctrl+Mayús+C), significa que deben pulsarse al mismo tiempo las tres teclas que hay delante y detrás del signo.

Abreviaturas:

  • Ctrl = tecla Control
  • Cmd = tecla Comando del teclado de Macintosh

Nota: Hay ciertos atajos de teclado que desempeñan diferentes funciones según la colocación del cursor (enfoque). Por ejemplo, si pulsa Inicio en Windows mientras esté visitando una página web, Netscape lo trasladará al principio de la página web. Sin embargo, si pulsa Inicio en Windows en un momento en el que el cursor esté en un campo de texto, el cursor irá al principio del campo de texto.

Sugerencia: en Window y Linux, se puede usar el teclado para entrar en las opciones de menú. Para abrir un menú del primer nivel, pulse Alt y la letra que esté subrayada en el nombre del menú. Después de abrir el menú, los submenús o los comandos se abren sin pulsar Alt. Por ejemplo, pulse Alt+B para abrir el menú Archivo.
