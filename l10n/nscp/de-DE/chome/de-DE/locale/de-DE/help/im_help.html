AOL Instant Messenger für Netscape

AOL Instant Messenger für Netscape ist eine Version der Software Instant Messenger von AOL, die in den Netscape-Browser integriert wurde. AOL Instant Messenger für Netscape ist mit allen anderen Versionen von AIM kompatibel. Mit dieser Software können Sie auf einfache Weise mit allen Ihren Freunden kommunizieren, die ebenfalls online sind. Folgendes ist mit dieser Anwendung möglich:

  • Einem Buddy, der online ist, Telegramme senden. Diese Nachrichten unterscheiden sich von normalen eMails dadurch, dass Ihre Nachricht sofort auf dem Bildschirm Ihres Buddy angezeigt wird, wenn er oder sie Instant Messenger auf dem PC installiert hat.
  • Eine Buddy-Liste mit Freunden erstellen, die online sind und an die Sie Telegramme senden möchten. Instant Messenger hilft Ihnen beim Auffinden Ihrer Freunde, damit Sie sie zu Ihrer Buddy-Liste hinzufügen können.
  • In Ihrem eigenen privaten Chat-Raum mit Freunden oder Ihrer Familie chatten.
  • Über die Privatsphärenfunktion für Buddies bestimmen, wer mit Ihnen kommunizieren kann.
  • Ihre Buddies nach eMail-Adresse suchen.
  • Benutzer, die Sie mit beleidigenden oder ärgerlichen Nachrichten belästigen, warnen.

Einrichten

In Netscape 7.0 können Sie einen einzigen Benutzernamen für viele Netscape-Funktionen verwenden, einschließlich AIM. Wenn Sie sich beim Starten von Netscape registriert haben, haben Sie bereits einen Benutzernamen und ein Passwort eingerichtet. Es sind keine weiteren Konfigurationsschritte erforderlich. Klicken Sie einfach im Menü Fenster auf Instant Messenger oder klicken Sie in Meine Seitenleiste auf die Registerkarte Buddy-Liste. Ihr Benutzername wird standardmäßig angezeigt. Sie müssen lediglich Ihr Passwort eingeben, das Sie bei der Registrierung eingerichtet haben. Damit ist die Software einsatzbereit.

Führen Sie die folgenden Schritte aus, falls Sie sich nicht registriert haben:

  1. Klicken Sie im Menü Fenster auf Instant Messenger.
  2. Klicken Sie auf Setup.
  3. Klicken Sie auf Ich habe keinen Benutzernamen. Ich muss einen neuen registrieren und dann auf Weiter.
  4. Geben Sie in das Dialogfeld, das geöffnet wird, den Namen ein, mit dem Sie sich bei Netscape angemeldet haben.
  5. Geben Sie den gewünschten Benutzernamen und das Passwort ein und klicken Sie dann auf Weiter.
  6. Geben Sie in das Dialogfeld, das angezeigt wird, Ihre eMail-Adresse, Ihr Geburtsdatum, Ihr Geschlecht und die PLZ ein und klicken Sie auf Weiter.
  7. Lesen Sie in dem Dialogfeld, das geöffnet wird, die Netscape-Vertraulichkeitsrichtlinien, klicken Sie auf I agree oder I disagree und klicken Sie dann auf Weiter.
  8. In dem Dialogfeld, das geöffnet wird, sehen Sie, ob der gewünschte Benutzername verfügbar ist. Sollte der Benutzername bereits vergeben sein, können Sie einen anderen Benutzernamen auswählen oder in einer Liste mit Vorschlägen, die Ihrer Eingabe ähneln, einen Benutzernamen auswählen. Klicken Sie auf Fertig stellen, wenn Ihr Benutzername akzeptiert wurde; Oder geben Sie einen neuen Benutzernamen ein oder wählen Sie einen der vorgeschlagenen Benutzernamen aus und klicken Sie dann auf Fertig stellen.

Anmerkung: Wenn Sie auf I disagree klicken, können Sie die Registrierung nicht vornehmen.

Kurze Zeit nach der Registrierung erhalten Sie eine Bestätigung per eMail. Sie müssen lediglich auf Antwort klicken und im Fenster Erstellungsbereich die Zeichenfolge OK eingeben. Sie erhalten dann eine zweite eMail mit Informationen zur Verwendung Ihres Benutzernamens.

Weitere Informationen zur Verwendung mehrerer Benutzernamen finden Sie unter Registrieren neuer Benutzernamen.

Anmelden bei AOL Instant Messenger

Bevor Sie mit Instant Messenger eine Nachricht senden oder chatten können, müssen Sie das Programm starten und sich anmelden.

So starten Sie Instant Messenger und melden sich an:

  1. Klicken Sie im Menü Fenster auf Instant Messenger.
  2. Geben Sie Ihren Benutzernamen in das Feld Benutzername ein. Falls Sie mehrere Benutzernamen verwenden, geben Sie den gewünschten Benutzernamen ein.
  3. Geben Sie Ihr Passwort in das Feld Passwort ein. Wenn Sie Ihr Passwort für die nächste Sitzung speichern möchten, klicken Sie auf Passwort speichern.
  4. Klicken Sie auf die Schaltfläche Anmelden bei Instant Messenger oder drücken Sie die Eingabetaste.

Tipp: Wenn Instant Messenger bei jedem Start von Netscape automatisch gestartet werden soll, aktivieren Sie im Anmeldungsbildschirm die Option Beim Start anmelden.

Anmerkung: Wenn Sie bereits AOL Instant Messenger separat oder über den AOL-Client verwendet und dann Netscape auf Ihrem Computer installiert haben, müssen Sie Ihre Einstellungen neu auswählen. Einige Einstellungen werden im Gegensatz zu Buddy-Listen nicht automatisch in der Netscape-Version übernommen. Informationen zur Neueingabe Ihrer Einstellungen finden Sie unter Festlegen der Einstellungen für Instant Messenger.

AOL Instant Messenger für Netscape ist so konfiguriert, dass die Anwendung beim Start des Computers automatisch gestartet wird. Führen Sie die folgenden Schritte aus, falls Sie dies ändern möchten:

  1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
  2. Klicken Sie auf die Kategorie Instant Messenger.
  3. Deaktivieren Sie das Kontrollkästchen Automatisch beim Systemstart starten.
  4. Klicken Sie auf OK.

Diese Option wird durch die Instant Messenger-Einstellung kontrolliert. Sie ist jedoch aktiv, unabhängig davon, ob Sie AIM für Netscape oder ICQ für Netscape verwenden. Mit dieser Option wird das AIM- oder ICQ-Fenster geöffnet. Sie werden jedoch nicht automatisch angemeldet, außer Sie aktivieren zusätzlich das Kontrollkästchen Beim Start anmelden in AIM bzw. Beim Starten verbinden in ICQ.

Anmerkung: Dieses Kontrollkästchen wird in Abhängigkeit von der verwendeten Plattform möglicherweise nicht angezeigt. Es ist nicht verfügbar, wenn Sie eine andere Version von AOL Instant Messenger verwenden oder der separate ICQ-Client so konfiguriert ist, dass er zusammen mit dem Computer automatisch gestartet wird.

Hinweise zur Buddy-Liste

Wenn Sie sich bei Instant Messenger anmelden, wird automatisch Ihre Buddy-Liste angezeigt. Im Fenster Buddy-Liste werden automatisch Ordner für drei Gruppen erstellt: Buddies, Familie und Kollegen. Fügen Sie Ihre Buddies zu einer dieser Gruppen hinzu oder erstellen Sie zusätzliche Gruppen.

Anmerkung: Die Buddy-Liste wird sowohl in der Instant Messenger-Anwendung selbst als auch in Meine Seitenleiste auf der Registerkarte Buddy-Liste angezeigt. Sie können auf die meisten Funktionen von Instant Messenger zugreifen, wenn Sie in Meine Seitenleiste auf die Registerkarte Buddy-Liste klicken. Sie müssen jedoch im Fenster Fenster auf Instant Messenger klicken, um die internen Menüs verwenden zu können.

Im Fenster Buddy-Liste sind zwei Registerkarten vorhanden: Online und Liste einrichten.

Die Registerkarte Online:

  • Die Registerkarte Online zeigt an, wenn Ihre Buddies online sind. Sobald sich einer Ihrer Buddies angemeldet hat, wird er in der Buddy-Liste angezeigt und Sie können ihm sofort ein Telegramm schicken.
  • Gruppen werden im Fenster Buddy-Liste zuerst mit den eingeblendeten Zweigen angezeigt, mit einer Auflistung aller Buddies in jeder Gruppe. Sie können zwischen der Anzeige mit den ein- oder ausgeblendeten Zweigen wählen, indem Sie auf den jeweiligen Gruppennamen doppelklicken oder den Gruppennamen auswählen (hervorheben) und die Eingabetaste drücken.
  • Rechts von jedem Gruppennamen wird in Klammern eine Zahl angezeigt, die angibt, wie viele Personen von allen Buddies dieser Gruppe gerade online sind. So bedeutet beispielsweise (3/9), dass drei von neun Buddies in dieser Gruppe derzeit online sind.
  • Namen, die grau oder kursiv angezeigt werden, geben die Buddies an, die sich gerade abgemeldet haben oder im Ruhemodus befinden.

Die Registerkarte Liste einrichten:

  • Jede Gruppe wird als Ordner angezeigt.
  • Die Buddies innerhalb der einzelnen Gruppen werden als Verzweigungen des Ordners angezeigt. Sie müssen die Anzeige der Gruppe erweitern, um die Buddies in dieser Gruppe anzuzeigen. Um eine Gruppe zu erweitern, doppelklicken Sie auf den Namen der Gruppe.
  • Sie können nicht sehen, welcher Ihrer Buddies online ist, wenn Sie die Registerkarte Liste einrichten anzeigen. Klicken Sie auf die Registerkarte Online, um festzustellen, welche Buddies online sind.
  • Mit Liste einrichten können Sie außerdem Buddies zur Liste hinzufügen und Gruppen für sie erstellen. Sie können Gruppen für bestimmte Interessengemeinschaften, beispielsweise Tennispartner, Kollegen im gleichen Büro, Nachbarn usw., einrichten und dann diesen Gruppen Buddies hinzufügen. Mit der Registerkarte Liste einrichten können Sie auch Buddies und Gruppen löschen.

Tipp: Buddy-Listen sollten auf einem Server und nicht auf dem Computer gespeichert werden. Dies bedeutet, dass Sie von jedem Ort aus auf Ihre Buddy-Listen zugreifen können, sobald Sie sich bei Instant Messenger angemeldet haben.

Hinzufügen und Suchen von Buddies

Sie können einen Freund sofort in Ihre Buddy-Liste aufnehmen, wenn Sie seinen Benutzernamen kennen.

Starten Sie im Fenster Buddy-Liste.

  1. Klicken Sie auf die Registerkarte Liste einrichten:
  2. Klicken Sie auf die Schaltfläche Buddy hinzufügen (die Schaltfläche kann anders aussehen, wenn Sie nicht das Theme "Modern" verwenden). Das Dialogfeld Buddy hinzufügen wird geöffnet.
  3. Geben Sie den Benutzernamen des Buddys ein, den Sie hinzufügen möchten. Denken Sie daran, dass nur der Benutzername des Buddys und nicht die vollständige eMail-Adresse eingegeben wird. Beispiel: "PeterWagner" und nicht "PeterWagner@aol.com".
  4. Wählen Sie die Gruppe aus, der Sie den Benutzernamen zuordnen möchten.
  5. Klicken Sie auf OK.

Wenn Sie den Benutzernamen eines Freundes nicht kennen, kann Ihnen AIM beim Auffinden dieses Freundes anhand der eMail-Adresse helfen.

Starten Sie im Fenster Buddy-Liste der separaten AIM-Version (nicht der Version in Meine Seitenleiste).

  1. Klicken Sie im Menü Leute auf Buddy suchen.
  2. Geben Sie die eMail-Adresse des Buddys in das Feld eMail-Adresse des Freundes ein und klicken Sie auf Weiter. Instant Messenger zeigt alle Benutzernamen für die von Ihnen angegebene eMail-Adresse an.
  3. Wählen Sie den Benutzernamen aus, den Sie zu Ihrer Liste hinzufügen möchten, und klicken Sie auf Buddy hinzufügen.
  4. Wählen Sie im Dialogfeld Buddy hinzufügen die Gruppe aus, zu der Sie den Buddy hinzufügen möchten, und klicken Sie auf OK.
  5. Klicken Sie anschließend auf Fertig stellen.

Überprüfen des Status von Buddies

Wenn Sie in der Buddy-Liste auf die Registerkarte Online klicken, können Sie sehen, welcher Ihrer Buddies online ist.

Wenn die Registerkarte Liste einrichten angezeigt wird, können Sie nicht sehen, welcher Ihrer Buddies online ist. Klicken Sie auf die Registerkarte Online.

Die Registerkarte Online:

  • Um eine Gruppe ein- und auszublenden, doppelklicken Sie auf den Namen der Gruppe.
  • Sie können anhand der Zahl in Klammern neben dem Gruppennamen sehen, wie viele Personen einer Gruppe online sind. So bedeutet beispielsweise (4/7), dass vier von sieben Buddies in der Gruppe derzeit online sind.
  • Wenn ein Buddy online ist, wird dessen Name in der Buddy-Liste mit dem Symbol für Online-Anwesenheit gekennzeichnet.
  • Grau dargestellte Namen sind inaktiv (inaktiv heißt, dass eine Person den Computer seit 15 Minuten nicht mehr verwendet hat, aber immer noch online ist) und ein kleines Uhrsymbol wird neben dem Namen angezeigt. Dieses Symbol wird nur angezeigt, wenn der Buddy die Meldung des inaktiven Zustands zulässt. Weitere Informationen finden Sie unter Deaktivieren der Inaktivitätsanzeige.
  • Wenn Ihr Buddy eine Abwesenheitsnachricht aktiviert hat, wird das Abwesenheitssymbol neben dem Namen angezeigt.

Sie können auch von Netscape eMail und Diskussionsforen aus herausfinden, welcher Ihrer Buddies gerade online ist. Wenn ein Buddy online ist und Sie bei Instant Messenger angemeldet sind, wird ein Anwesenheitssymbol neben dem Namen in den Feldern An, Von oder CC angezeigt. Dieses Symbol wird auch neben den Namen von Personen angezeigt, die nicht zu Ihren Buddies gehören. Nicht-Buddies sind Benutzer von Instant Messenger, die Sie nicht zu Ihrer Buddy-Liste hinzugefügt haben. Das Anwesenheitssymbol wird sowohl bei Buddies als auch bei Nicht-Buddies nur für die Benutzer angezeigt, die registriert sind und ihre Registrierung bestätigt haben.

Das Anwesenheitssymbol neben dem Namen eines Buddies verschwindet, wenn er sich an- oder abmeldet.

Einrichten von Buddy-Warnungen

  2. Klicken Sie im Menü Leute auf Hinweis wenn Buddy ist verfügbar. Das Dialogfeld Buddy-Benachrichtigung bearbeiten wird angezeigt.
  3. Aktivieren Sie die gewünschten Optionen: Benachrichtigen Sie mich, wenn Buddy... online ist, aus dem Ruhezustand zurückkehrt, aus dem Abwesenheitszustand zurückkehrt.
  4. Aktivieren Sie Benachrichtigen Sie mich durch: Anzeige des Fensters 'Buddy-Benachrichtigung' oder Benachrichtigen Sie mich durch: Spielen einer Tonfolge.

  1. Klicken Sie im Menü Datei auf Buddy-Liste exportieren.

Verwenden der Buddy-Symbole

Wenn Sie ein Telegramm senden, können Sie Ihr Telegramm (oder Nachrichten von bestimmten Buddies) mit Buddy-Symbolen ausschmücken. Warnen eines Benutzers

Wenn Sie einen Benutzer, der Ihnen eine beleidigende Nachricht gesendet hat, warnen möchten, klicken Sie in dem Fenster Telegramm, in dem Sie die Nachricht erhalten haben, auf Warnen.

Durch Warnungen werden die Möglichkeiten des verwarnten Absenders, Nachrichten zu senden und zu empfangen, eingeschränkt. Im Falle häufiger Warnungen kann der Benutzer keinerlei Nachrichten mehr mit anderen Benutzern austauschen. Erst nach einer "Abkühlungsphase" kann dieser Benutzer wieder Nachrichten senden und empfangen.

Wenn Sie auf Warnen klicken, können Sie auswählen, ob Sie die Warnung anonym aussprechen möchten. Anonyme Warnungen haben weniger Gewicht und der gewarnte Benutzer weiß nicht, wer die Warnung ausgegeben hat.

Für Warnungen, die Sie senden und erhalten, gilt Folgendes:

  • Sie können für jeweils eine Nachricht, die Ihnen ein Benutzer sendet, nur eine Warnung ausgeben.
  • Wenn ein Benutzer Sie warnt, sollten Sie am besten die Konversation beenden. Jede weitere Nachricht, die Sie senden, gibt dem anderen Benutzer die Möglichkeit, Sie erneut zu warnen.

Alternativ können Sie eine oder mehrere Personen ständig blockieren, sodass diese Ihnen keine Nachrichten mehr senden können. Weitere Informationen hierzu finden Sie unter Blockieren eines Benutzers.

Sie können sich nicht anmelden

Überprüfen Sie die folgenden Punkte:

  • Ist das eingegebene Passwort gültig? Beim Passwort wird die Groß- und Kleinschreibung berücksichtigt. Geben Sie das Passwort unter Berücksichtigung der Groß- und Kleinschreibung erneut ein und wiederholen Sie die Anmeldung.
  • Wurde Ihre Anmeldeanfrage abgelehnt? Wenn ja, haben Sie sich innerhalb eines kurzen Zeitraums zu häufig angemeldet. Warten Sie ein paar Minuten und versuchen Sie es noch einmal. Weitere Anmeldungsversuche verlängern die Wartezeit.
  • Sind Sie mit dem Internet verbunden? Wenn dies nicht der Fall ist, obwohl Sie zuvor die Verbindung herstellen konnten, überprüfen Sie Folgendes:
  • Funktioniert die Verbindung zu Ihrem Internetdienstanbieter (ISP)? Wenn nicht, wenden Sie sich an den Provider. Möglicherweise müssen Sie die Verbindungskonfiguration ändern.
  • Funktioniert die LAN-Verbindung ordnungsgemäß? Wenn nicht, wenden Sie sich an den Netzwerkadministrator. Möglicherweise müssen Sie die Verbindungskonfiguration ändern.

In folgenden Fällen kann die Anmeldung fehlschlagen:

  • Es besteht ein DNS-Problem bei Ihrem ISP oder in Ihrem Netzwerk. Wenden Sie sich an den ISP oder den Systemadministrator oder versuchen sie es später noch einmal.
  • Ihr Netzwerk verwendet eine Firewall. Wenden Sie sich in diesem Falle an Ihren Systemadministrator, um Informationen zum Einrichten eines Proxy-Servers zu erhalten. Diese Informationen müssen Sie in den Verbindungseinstellungen eingeben.
  • Der Instant Messenger-Dienst ist vorübergehend nicht verfügbar. Versuchen Sie es später noch einmal.

Wenn keine der oben genannten Bedingungen zutrifft, besteht wahrscheinlich ein vorübergehendes Problem im Netzwerk, das die Verbindung verhindert. Sie müssen lediglich warten und es später noch einmal versuchen.

Überprüfen Sie Folgendes:

Um die Option Alle Benutzer blockieren zu deaktivieren, klicken Sie im Menü Bearbeiten auf Einstellungen. Klicken Sie unter Instant Messenger auf die Kategorie Privatsphäre und deaktivieren Sie diese Option.
  • Werden Sie von allen Ihren Buddies blockiert? Wenn ja, klicken Sie im Menü Leute auf Freund anmelden, um einen Freund zur Registrierung beim Instant Messenger-Dienst einzuladen.

Einstellungen für Instant Messenger – Allgemein

In diesem Abschnitt wird beschrieben, wie Sie im Fenster Einstellungen für Instant Messenger allgemeine Richtlinien für Nachrichten festlegen. Wenn Sie dieses Fenster noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

  1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
  2. Klicken Sie auf die Kategorie Instant Messenger.

Legen Sie im Abschnitt Anmelden bei Instant Messenger dieses Fensters die folgenden Einstellungen fest:

  • Automatisch beim Systemstart starten: Instant Messenger wird automatisch gestartet, wenn Sie den Computer starten. Deaktivieren Sie diese Option, falls Sie dies nicht wünschen. Diese Option ist nur verfügbar, wenn sie von Ihrem System unterstützt wird und wenn Sie noch keinen separaten ICQ-Client oder einen anderen AOL-IM-Client so konfiguriert haben, dass diese zusammen mit dem Computer gestartet werden.

    Diese Option wird durch die Instant Messenger-Einstellung kontrolliert. Sie ist jedoch aktiv, unabhängig davon, ob Sie AIM für Netscape oder ICQ für Netscape verwenden. Mit dieser Option wird das AIM- oder ICQ-Fenster geöffnet. Sie werden jedoch nicht automatisch angemeldet, außer Sie aktivieren das Kontrollkästchen Beim Start anmelden in AIM bzw. Beim Start verbinden in ICQ.

  • Bei Anmeldung Fenster 'Heute' anzeigen: Diese Option öffnet den Heute-Minibrowser automatisch, anstatt damit zu warten, bis Sie auf die Schaltfläche Heute klicken.
  • Automatisches Anmelden bei Websiten über den Benutzernamen-Service deaktivieren: Diese Option deaktiviert die Standardeinstellung, dass Sie automatisch bei Websites angemeldet werden, die für die Verwendung Ihres AOL-Benutzernamens und -Passworts eingerichtet wurden.

Legen Sie im Abschnitt Nachrichten erstellen und formatieren dieses Fensters die folgenden Einstellungen fest:

  • Eingabetaste fügt einen Zeilenumbruch ein: Wenn mit der Eingabetaste ein Zeilenumbruch eingefügt werden soll, aktivieren Sie diese Option. Deaktivieren Sie diese Option, wenn beim Drücken der Eingabetaste die Nachricht gesendet werden soll.
  • Tabulatortaste fügt einen Tabulator ein: Wenn mit der Tabulatortaste ein Tabulator eingefügt werden soll, aktivieren Sie diese Option. Deaktivieren Sie diese Option, wenn Sie mit der Tabulatortaste im Fenster Telegramm von einem Feld zum anderen wechseln möchten.
  • Zeitanzeige auf allen Nachrichten anzeigen: Wenn alle Nachrichten, die Sie senden und empfangen, mit der Sendezeit markiert werden soll, aktivieren Sie diese Option. Beachten Sie, dass die Verwendung von Zeitanzeigen für einzelne Nachrichten deaktiviert werden kann. Diese Einstellung steuert die standardmäßige Verwendung von Zeitanzeigen.
  • Emoticons als Grafiken anzeigen: Mit dieser Option bestimmen Sie, ob Emoticons (oder grafische Smileys) in Ihren Chat-Nachrichten oder Telegrammen angezeigt werden.

Legen Sie im Abschnitt eMails & Diskussionsforen dieses Fensters die folgende Einstellung fest:

  • Online-Zustand auf dem Nachrichtenumschlag anzeigen: Wenn das Anwesenheitssymbol neben allen Buddies, die online sind, auf dem Umschlag einer Netscape eMail & Diskussionsforen-Nachricht angezeigt werden soll, aktivieren Sie diese Option.

Einstellungen für Instant Messenger – Privatsphäre

In diesem Abschnitt wird die Verwendung des Instant Messenger-Fensters Einstellungen für Privatsphäre beschrieben. Wenn Sie dieses Fenster noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

  1. Melden Sie sich bei AIM an. (Sie müssen angemeldet sein, um die Einstellungen für die Privatsphäre ändern zu können.)
  2. Klicken Sie im Menü Bearbeiten auf Einstellungen.
  3. Klicken Sie in der Kategorie Instant Messenger auf Privatsphäre. (Wenn in dieser Kategorie keine Einträge angezeigt werden, doppelklicken Sie auf Instant Messenger, um die Liste zu erweitern.)

Klicken Sie im Abschnitt Wer kann mich kontaktieren dieses Fensters auf das Optionsfeld oder das Kontrollkästchen der zur Auswahl stehenden Einstellungen:

  • Alle Benutzer zulassen: Wählen Sie diese Option, wenn Sie keine Beschränkungen für das Senden von Nachrichten aufstellen möchten.
  • Nur Benutzern in Buddy-Liste die Verbindung erlauben: Wählen Sie diese Option, wenn Sie nur von Personen, die in Ihrer Buddy-Liste vorhanden sind, Nachrichten empfangen möchten.
  • Diesen Benutzern erlauben: Wählen Sie diese Option, um die Personen festzulegen, von denen Sie Nachrichten empfangen möchten. Klicken Sie auf Namen hinzufügen und geben Sie die Benutzernamen der gewünschten Personen ein.
  • Alle Benutzer blockieren: Wählen Sie diese Option, wenn Sie überhaupt keine Nachrichten empfangen möchten.
  • Diese Benutzer blockieren: Wählen Sie diese Option, wenn Sie von bestimmten Personen keine Nachrichten empfangen möchten. Klicken Sie auf Namen hinzufügen und geben Sie die Benutzernamen der gewünschten Personen ein.
  • Alle eingehenden Chat-Einladungen blockieren: Wählen Sie diese Option, wenn Sie verhindern möchten, dass Chat-Einladungen auf dem Bildschirm angezeigt werden.
  • Andere Benutzer können sehen, wie lange ich inaktiv war: Wählen Sie diese Option, wenn die Benutzer sehen sollen, dass Sie die Tastatur nicht verwendet haben (inaktiv) und wie lange Sie die Tastatur nicht verwendet haben.

Wählen Sie im Abschnitt Dies sollen Benutzer, die meine eMail-Adresse kennen, sehen dieses Fensters eine der folgenden Optionen in der Dropdown-Liste aus:

  • Meinen Benutzernamen: Wählen Sie diese Option, wenn für Benutzer, die Ihre eMail-Adresse kennen, über die Funktion Buddy suchen Ihr Benutzername angezeigt werden sollen.
  • Nur, dass ich über ein Konto verfüge: Wählen Sie diese Option, wenn Benutzer, die Ihre eMail-Adresse kennen, über die Funktion Buddy suchen nur sehen sollen, dass Sie ein Benutzerkonto haben.
  • Nichts über mich: Wählen Sie diese Option, wenn Benutzer, die Ihre eMail-Adresse kennen, über die Funktion Buddy suchen nichts über Sie erfahren sollen.

Einstellungen für Instant Messenger – Buddy-Symbole

In diesem Abschnitt wird die Verwendung des Instant Messenger-Fensters Einstellungen für Buddy-Symbole beschrieben. Wenn Sie dieses Fenster noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

  1. Melden Sie sich bei AIM an. (Sie müssen angemeldet sein, um die Einstellungen für die Buddy-Symbole ändern zu können.)
  2. Klicken Sie im Menü Bearbeiten auf Einstellungen.
  3. Klicken Sie in der Kategorie Instant Messenger auf Buddy-Symbole. (Wenn in dieser Kategorie keine Einträge angezeigt werden, doppelklicken Sie auf Instant Messenger, um die Liste zu erweitern.)

Im Fenster Einstellungen für Buddy-Symbole können Sie ein Buddy-Symbol auswählen, das allen Ihren Telegrammen hinzugefügt wird. Außerdem können Sie in diesem Fenster ein Buddy-Symbol für Buddies in Ihrer Buddy-Liste auswählen. Schließlich können Sie damit Ihre Buddy-Symbol-Eigenschaften als Empfänger einrichten.

So richten Sie Ihre Buddy-Symbol-Einstellungen als Absender ein:

  1. Wählen Sie in einer der im Fenster Einstellungen angezeigten Kategorien ein Buddy-Symbol aus.
  2. Oder suchen Sie auf der Festplatte nach einer Grafik, die Sie verwenden möchten.

Grafiken mit den folgenden Formaten können verwendet werden: GIF, JPG, BMP, ICO oder XBM. Die Grafiken dürfen eine Größe von maximal 7.168 Bytes aufweisen. Unabhängig davon, ob Sie eine der vorgefertigten Grafiken auswählen oder eine eigene Grafik verwenden, werden diese skaliert und an das kleine Quadrat in der linken oberen Ecke des Fensters angepasst. Beim Senden eines Telegramms wird das Symbol in der linken unteren Ecke des Nachrichtenfensters angezeigt.

So richten Sie ein Buddy-Symbol für eine Person in Ihrer Buddy-Liste ein:

  1. Wählen Sie den Namen des Buddys aus, für den Sie ein Buddy-Symbol einrichten möchten.
  2. Wählen Sie in einer der im Fenster Einstellungen angezeigten Kategorien ein Buddy-Symbol aus.
  3. Oder suchen Sie auf der Festplatte nach einer Grafik, die Sie verwenden möchten.

Grafiken mit den folgenden Formaten können verwendet werden: GIF, JPG, BMP, ICO oder XBM. Die Grafiken dürfen eine Größe von maximal 7.168 Bytes aufweisen. Unabhängig davon, ob Sie eine der vorgefertigten Grafiken auswählen oder eine eigene Grafik verwenden, werden diese skaliert und an das kleine Quadrat in der linken oberen Ecke des Fensters angepasst. Beim Senden eines Telegramms wird das Symbol in der linken unteren Ecke des Nachrichtenfensters angezeigt.

Anmerkung: Das zugewiesene Buddy-Symbol überschreibt etwaige Buddy-Symbole, die der Buddy festgelegt hat. Wenn Sie von diesem Buddy eine Nachricht erhalten, wird das Buddy-Symbol angezeigt, das Sie zugewiesen haben. Buddy-Symbole, die der Buddy zugewiesen hat, werden nicht angezeigt.

So richten Sie Ihre Buddy-Symbol-Einstellungen als Empfänger ein:

  • Legen Sie unter Buddy-Symbole anzeigen fest, ob Buddy-Symbole von allen Personen in der Buddy-Liste akzeptiert werden sollen, ob alle Buddy-Symbole abgelehnt werden sollen oder ob Buddy-Symbole zwar akzeptiert, aber nicht angezeigt werden sollen.
  • Alternativ können Sie auch das Kontrollkästchen Anklopf-Dialogfeld anzeigen aktivieren.

Wenn Sie Anklopf-Dialogfeld anzeigen aktivieren, wird jedes Mal, wenn ein Buddy Ihnen eine Nachricht mit einem Buddy-Symbol sendet, ein Dialogfeld angezeigt, in dem Sie von Fall zu Fall das Symbol akzeptieren oder ablehnen können.

Anmerkung: Das Ablehnen von Buddy-Symbolen hat nichts mit dem Blockieren von Benutzern zu tun. Selbst wenn Sie alle Buddy-Symbole ablehnen, können Sie weiterhin Telegramme von allen nicht-blockierten Buddies empfangen. Sie werden lediglich ohne die Symbole angezeigt.

Einstellungen für Instant Messenger – Verbindung

In diesem Abschnitt wird die Verwendung des Instant Messenger-Fensters Einstellungen für Verbindung beschrieben. Wenn Sie dieses Fenster noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

  1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
  2. Klicken Sie in der Kategorie Instant Messenger auf Verbindung. (Wenn keine Unterkategorien angezeigt werden, doppelklicken Sie auf Instant Messenger, um die Liste zu erweitern.)

Wenn Sie Internetverbindungen über einen Internetdienstanbieter herstellen, müssen Sie höchstwahrscheinlich keine Änderungen in diesem Fenster vornehmen. Wenn Sie Internetverbindungen über eine LAN-Firewall herstellen (z. B. am Arbeitsplatz), benötigen Sie möglicherweise vom Netzwerkadministrator Informationen zum Konfigurieren des Proxyabschnitts in diesem Fenster.

Host

In diesem Feld ist die Standardinternetadresse des in Netscape integrierten Instant Messaging-Servers enthalten, der AOL Instant Messenger für Netscape und ICQ für Netscape unterstützt. Normalerweise müssen Sie dieses Feld nicht ändern. Wenn der Proxyserver den Hostnamen auflösen muss, müssen Sie möglicherweise die IP-Adresse anstelle des Servernamens in dieses Feld eingeben. Die IP-Adresse erhalten Sie von Ihrem Netzwerkadministrator.

Port

In diesem Feld ist die Standard-Serverportnummer, die der in Netscape integrierte Instant Messaging-Server verwenden soll, enthalten. Normalerweise müssen Sie dieses Feld nicht ändern.

Verbinden mit Proxy

Wenn Sie dieses Kontrollkästchen aktivieren, wird durch die Instant Messenger-Software die Proxyserversteuerung aktiviert. Besorgen Sie sich die folgenden Informationen vom Netzwerkadministrator und geben Sie sie ein.

Host

Geben Sie den vom Netzwerkadministrator angegebenen Namen des Proxyservers ein.

Port

Geben Sie die vom Netzwerkadministrator angegebene Portnummer ein.

Benutzername

Geben Sie den vom Netzwerkadministrator angegebenen Authentifizierungs-Benutzernamen ein.

Passwort

Geben Sie das vom Netzwerkadministrator angegebene Authentifizierungs-Passwort ein.

Protokoll

Geben Sie das vom Netzwerkadministrator angegebene Protokoll ein: SOCKS 4, SOCKS 5, HTTPS oder HTTP.
