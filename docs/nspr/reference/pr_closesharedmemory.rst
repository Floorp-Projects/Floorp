PR_CloseSharedMemory
====================

Closes a shared memory segment identified by name.


Syntax
------

.. code:: eval

   #include <prshm.h>

   NSPR_API( PRStatus )
     PR_CloseSharedMemory(
        PRSharedMemory *shm
   );


Parameter
~~~~~~~~~

The function has these parameter:

shm
   The handle returned from :ref:`PR_OpenSharedMemory`.


Returns
~~~~~~~

:ref:`PRStatus`.
