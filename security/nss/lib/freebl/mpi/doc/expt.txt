Exponentiation

For exponentiation, the MPI library uses a simple and fairly standard
square-and-multiply method.  The algorithm is this:

Input:	a, b
Output: a ** b

	s = 1

	while(b != 0)
	  if(b is odd)
	    s = s * a
	  endif

	  b = b / 2

	  x = x * x
	endwhile

	return s

The modular exponentiation is done the same way, except replacing:

	s = s * a

with
	s = (s * a) mod m

and replacing

	x = x * x

with

	x = (x * x) mod m
