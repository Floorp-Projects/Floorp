speculativeConnect

abstract fun speculativeConnect(url: String): Unit (source)

Opens a speculative connection to the host of url.

This is useful if an app thinks it may be making a request to that host in the near future. If no request is made, the connection will be cleaned up after an unspecified.

Not all Engine implementations may actually implement this.
