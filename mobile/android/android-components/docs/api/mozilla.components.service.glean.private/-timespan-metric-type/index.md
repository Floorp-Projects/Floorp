# TimespanMetricType

`data class TimespanMetricType : `[`CommonMetricData`](../-common-metric-data/index.md) [(source)](https://github.com/mozilla-mobile/android-components/blob/master/components/service/glean/src/main/java/mozilla/components/service/glean/private/TimespanMetricType.kt#L23)

This implements the developer facing API for recording timespans.

Instances of this class type are automatically generated by the parsers at build time,
allowing developers to record values that were previously registered in the metrics.yaml file.

The timespan API exposes the [start](start.md), [stop](stop.md) and [cancel](cancel.md) methods.
