Overrides EngineSession.enableTrackingProtection

See EngineSession.enableTrackingProtection

Note that specifying tracking protection policies at run-time is not supported by SystemEngine. Tracking protection is always active for all URLs provided in domain_blacklist.json and domain_overrides.json, which both support specifying categories. See UrlMatcher for how to enable/disable specific categories.
