Help with Rhino

Have a question that you can't find answer to in the documentation? Here are some additional resources for help:
 

Newsgroup

The netscape.public.mozilla.jseng newsgroup answers questions about both Rhino and the C implementation of JavaScript.
 

Bug System

Use Bugzilla to enter bugs against Rhino. Note that Rhino has its own product category.
 

Module Owner

The module owner, , can be mailed for help as well, although he may copy his response to the newsgroup to help others.
